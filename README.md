# 潇龙霞八字计算器 - 第一阶段

纯前端八字计算工具，无需后端，打开即用。

## 功能特点

- ✅ 四柱计算（年柱、月柱、日柱、时柱）
- ✅ 天干地支 + 五行显示
- ✅ 十神分析
- ✅ 纳音五行
- ✅ 五行统计
- ✅ 响应式设计（支持手机/电脑）

## 使用方法

1. 直接在浏览器打开 `index.html`
2. 输入出生年月日时
3. 点击"开始测算"
4. 查看八字命盘

## 技术栈

- HTML5 + CSS3 + JavaScript（纯前端）
- 无需后端服务器
- 无需安装依赖

## 算法说明

- **年柱**：基于年份推算天干地支
- **月柱**：年上起月法（甲己之年丙作首...）
- **日柱**：基于1900年1月1日甲子日推算
- **时柱**：日上起时法（甲己还加甲...）
