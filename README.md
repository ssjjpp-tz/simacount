## 🚀 使用方法

1. 访问版本选择页
2. 选择要使用的版本（推荐 v2.0）
3. 填写出生信息：年月日时 + 性别
4. 填写测算日期（默认今天）
5. 点击"开始测算"
6. 查看八字命盘、今日运势、黄历宜忌
