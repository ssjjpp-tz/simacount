# 潇龙霞八字黄历 v2.0

八字命理 + 日运分析 + 黄历宜忌，纯前端实现。

## 在线访问

🔗 **https://ssjjpp-tz.github.io/simacount/**

## 功能特点

### 基础功能
- ✅ 四柱八字计算（年柱、月柱、日柱、时柱）
- ✅ 天干地支 + 五行显示
- ✅ 十神关系分析
- ✅ 纳音五行

### v2.0 新增功能
- ✅ **测算日期选择** - 可算任意日期的运势
- ✅ **黄历宜忌** - 今日适合/不适合做的事
- ✅ **日运分析** - 综合评分、事业运、财运
- ✅ **八字关系** - 当日天干与你的关系（比肩、食神、正官等）
- ✅ **详细建议** - 根据八字给出的具体行动建议

## 使用方法

1. 填写**出生信息**：年月日时 + 性别
2. 填写**测算日期**：要查看运势的日期（默认今天）
3. 选择**所在城市**（可选，用于时辰校准）
4. 点击"开始测算"
5. 查看八字命盘、今日运势、黄历宜忌

## 技术栈

- HTML5 + CSS3 + JavaScript
- 纯前端，无需后端
- GitHub Pages 部署

## 算法说明

### 八字计算
- 年柱：基于年份推算
- 月柱：年上起月法
- 日柱：基于1900年甲子日推算
- 时柱：日上起时法

### 黄历宜忌
- 十二建星（建除十二神）
- 二十八宿
- 综合算法生成宜忌事项

### 日运分析
- 日干与当日天干的关系（十神）
- 五行生克分析
- 综合评分算法
